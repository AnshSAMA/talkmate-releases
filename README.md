# TalkMate Releases

This public repository hosts TalkMate's Sparkle update feed, packaged update
archives, and release notes.

The TalkMate source repository remains private. Never add source code,
credentials, private signing keys, customer information, or internal builds
here.

## Update Feed

The update feed will be available through GitHub Pages at:

`https://anshsama.github.io/talkmate-releases/appcast.xml`

Release archives must be signed with TalkMate's private Sparkle EdDSA key before
they are published. The private key must remain outside this repository.
